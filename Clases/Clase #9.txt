-Estructura de datos Linear
Que es Abstract Data Types?
Es un set de objetos juntos que forman un set de operaciones
Son abstracciones matematicas, no incluye manera de implementar operaciones

-Tipo de datos primitivos(no se pueden descoimponer)
	Enteros
	Reales
	Booleanos

-Tipo de estructura de datos ADT
	Listas
	Colas
	Sets 
	Graficos
Tienen operaciones asosiadas
Añadir / Add
Remover / Remove
Contener / Contains

-Como implementar en JAVA?
Utilizando clases y metodos

En OO, ADT pueden ser iguales a una interface o clase con otras caracteristicas

-Que operaciones debe tener?
Se construye como lo quiera definir el programador

-Listas enlazadas
Es como un array pero no tiene un tamaño definido e insertar valores de manera dinamica
Un modo contiene datos y una referencia a otro nodo, asi es como estan conectados el ultimo nodo apunta a "null" como nodo
de referencia
	(Cabeza de lista) A1---->A2---->A3---->null

Se puede apuntar de la cabeza al ultimo nodo para mayor eficiencia

-Doblemente enlazada (variacion de lista simple)
Se puede ir en ambas direcciones 
El nodo tiene referencia a un nodo anterior y uno siguiente
Tienen mas conecciones debido a el previo y el siguiente
	(Cabeza de lista) A1<---->A2<---->A3<---->null
